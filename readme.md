# Um jogo para Devs.

## Sobre
O projeto foi desenvolvido a fim de criar uma página que apresente um produto para auxiliar o aprendizado de pessoas desenvolvedoras. Assim, idealizamos o Batstrap, que consiste num jogo desenvolvido com o objetivo de auxiliar a jornada de aprendizado de dev’s no bootstrap. Sendo que no jogo levamos o morcego Bruce até a sua caverna utilizando bootstrap, movimentando-o através de métodos como classes containers e sistema grid. Este é o Batstrap, para conhecer mais acesse nossa página!
